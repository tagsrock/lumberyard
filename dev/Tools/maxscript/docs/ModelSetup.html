How to set up models in the main dialogs Misc rollout

This tutorial describes how to set up models and how to maintain them.

In this example I choosed a simple charachter.

1. Open the dialog and hit Load Model:


 2. Select Edit Entries:


 3. A dialog should pop up. In the list, double click in an empty line to add a model:


 4. In the next window you must select the model you want to add:


 5. Now the model should be in the list. To maintain them, double click on the line with the model or click once to rename the entry name:


 6. When the list is saved, the new entry is available when clicking again on Load Model:
